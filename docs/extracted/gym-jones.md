## Philosophy

- "We value horsepower ahead of appearance — NOT VICE-VERSA." (Twight)
- The mind is the primary limiter. Work tolerance is the most trainable characteristic.
- "Fit for whatever comes next" — maintain 60–70% across all fitness domains rather than peak in one
- Very few operators need complex sport-specific training; they need functional movement and systemic endurance
- Excessive strength/speed development increases injury risk; power-endurance, strength-endurance, and general endurance are more valuable in operational context
- There are no shortcuts

## Methodology

**6-day training week:**
- 3 key workouts (Strength, Circuit, Accumulation)
- 2 recovery-based workouts (low intensity, restorative)
- 1 endurance workout (long duration, terrain-based)
- 1 full rest day
- Programs last 1 month and can be repeated or cycled 2–3x

**Intensity distribution:** even mix of strength (neural), power-endurance, muscular endurance, systemic endurance (no polarized model)

**Hard/Easy pattern:** every hard day followed by recovery day

**Progression levers:** reduce rest periods (bodyweight), increase weight (loaded), NOT both simultaneously

## Workout Archetypes

### Strength Day
1. Warm-up: wall squats (2×5), air squats (2×10), goblet squats (2×5 @ 25#), single-leg squats (5×5 @ 18–20" box)
2. Barbell or DB complex (3–4 sets)
   - Six-Way Barbell Complex: DL → Bent Row → Hang Clean → Front Squat → Push Press → Back Squat → Push-up (6 reps each, progressive load: 75#–85#–95#–105#, 1–2 min rest)
   - DB Complex: Hi-Pull → Row → Hang Clean → Front Squat Push Press → OHS/SOTS → Push-up/Row (6 reps each, 20–25# DBs)
3. Primary strength (80% 1RM): Deadlift 5×3, Front Squat 10×2 @ 1-min rest, or OHS 5×3 (3–4 min rest between sets)
4. Core: TGU 6× @ 25# + 60-sec FLR (5 rounds) OR pull-up ladder 5×(1–6)
5. Finisher: 1000m–2000m row for time

### Circuit Day (mixed modalities, higher volume)
- DB complex (3 sets) → medium-rep circuits (e.g. ball slam + KB step-up: 30–20–10) → density intervals (4×20sec/10sec row) → 300-sec FLR

### Accumulation Day (density/high-rep)
- Tempo work (e.g. clean triplet every 30 sec for 10 min) → high-rep circuits ("Those Burpees Suck": 10 pull-ups → 20 KB swings @ 53# → 30 box jumps → 40 push-ups → 50 sit-ups → 60 burpees)
- Single-leg: SLDL 3×10 @ 95–135#, lunges 2×20m
- Static holds: 300-sec FLR (accumulated), 90-sec overhead hold, 300-sec dead hang

### Recovery Day
- Format A: 60–120 min easy row/run/swim/hike (conversational pace)
- Format B: High-rep/low-intensity (e.g. 100× TGU @ 15#, 3×20 DL @ 30% 1RM)
- Endurance hike: 90–120 min with elevation, 20–30# pack

### Power-Endurance Day
- Warm-up: 10-min easy run
- Time trial: 3-mile run all-out OR 400m progressive intervals
- Loaded variant: 5 lunges + 5 push press while holding 45# bar, continuing 400m

### Interval/Threshold Day
- Warm-up: 10-min easy
- Examples:
  - 6× 500m row @ ~15 sec slower than PR pace, 3-min active rest (1:2 work:rest)
  - 15× (30-sec row / 90-sec rest) with distance incrementing 1m per set (150m→165m)
  - 5× 1-min all-out sprints, 4-min active recovery
  - 4× 800 yards at 10% faster than 2-mile PR pace
- Cool-down: 10–20 min easy

### Breathing Ladder
- KB Swing (25# → 35# → 44# → 53# warm-up progression)
- 1–20–1 or 3×(1–20) KB Swing @ 53# breathing ladder
- Athlete performs reps with prescribed breaths between; continues until can no longer complete round

## Exercises

**Squat patterns:** air squat, wall squat, goblet squat (25–60#), jump squat, tuck jump, single-leg squat @ box (18–20"), deck squat

**Deadlift patterns:** conventional DL (primary, 50–90% 1RM), SLDL @ 95–135#, DL @ 30% for high-rep recovery

**Upper push:** push-up, weighted push-up, push press, dumbbell push press, floor bench press, OHS, SOTS press

**Upper pull:** pull-up (ladders 1–6, 50+ reps), bent-over row, one-arm row, ring dip (high-rep circuits), dead hang (300-sec static)

**Complex sequences:** Six-Way Barbell Complex, DB Complex, Man-Maker, Burpee

**Loaded carries:** farmer carry, suitcase carry, bear crawl (weighted DBs)

**Lower accessory:** walking lunge (forward/backward), step-up (up to 95–135#), box jump (20–24"), split jump, frog hop

**Core:** TGU @ 15–25# (100-rep OR 5–6 rep strength), FLR (300-sec), overhead hold (300-sec), mountain climber, sit-up, V-sit, flutter kick

**Kettlebell:** KB swing (25#–53# progression), KB snatch (single arm), KB jerk/push press, KB figure-8

**Endurance:** running (easy/tempo/intervals/sprints), rowing (500m–2000m pieces), hiking (pack weight 20–30#)

## Performance Benchmarks

- Deadlift: Test 1RM; training percentages built from it (80% common)
- Front Squat: Test 1RM
- OHS: Test 1RM
- 500m row: Establish PR pace; interval training at PR+15 sec
- 3-mile run: All-out time trial
- 2-mile PR pace: Referenced for 800-yd interval speeds

## Programming Rules

| Rule | Detail |
|------|--------|
| Session length | 60–90 min key workouts; 60–120 min recovery |
| Rest (strength) | 3–5 min between heavy sets |
| Rest (complexes) | 1–2 min between sets |
| Rest (metabolic) | Structured 20–30 sec patterns |
| Load progression | Barbell complex: 75→85→95→105#; KB: 25→35→44→53# |
| Beginner mods | Push-ups on knees; plank pulls for pull-ups; lighter loads |
| Cycling | Repeat program 2–3× or alternate Operator I / Operator II |

## Equipment Required

- Open space (pull-up bar)
- Box/bench (20–24")
- Dumbbells (15–60#)
- Barbell with plates (95–135# minimum for most workouts)
- Rower or other low-impact cardio machine
- Kettlebells (25–53#)
- Medicine ball (20–25#)
- ~80% of workouts require no special equipment
