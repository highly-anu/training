## Philosophy

- "Elite level of fitness over a broad base" — not specialized in any single domain
- Need to function at high level across: heavy lifting, sprinting, extreme environment operations
- Durability over peak performance; repeatable output under fatigue
- Simple equipment: barbell, KB, sandbag, bodyweight, rower

## Workout Type Definitions

### Warm-Up Formats
- **Barbell Complex** (4 rounds, progressive: 75#–85#–95#–105#): DL → Row → Hang Clean → Front Squat → Push Press → Back Squat → Push-up (6 reps each, no rest)
- **12-min KB Complex** @ 35#: 1-arm swing R/L, figure-8 both directions, clean & press R/L, slasher-to-halo ×2, snatch R/L, lunge with KB R/L (1 min each, no breaks)
- **Sandbag Get-ups** 20× each shoulder @ 80#, 5 rounds
- **Wall Ball + Sit-up** 10×/20×, 5 rounds
- **KB Swing + Jumping Lunge + Push-up** AMRAP variants

### Power Endurance
Workouts designed for lactic capacity — combining loaded carries/lifts with running:
- "Counting Dead Bodies" — 5 lunges + 5 push press while holding 45# bar for 400m (20–30 min)
- "Get-ups Till You Throw Up" — 100 sandbag get-ups @ 85–95# then suicide sprint every 30 sec for 10 min
- "Tabata Hell" — Tabata squats, push-ups, ball slams @ 20#, sit-ups, KB swings @ 53# (1-min rest between; score = total reps)
- "Teabag Your Buddy Tonight" — EMOM 20 min: 2× sandbag clean & squat @ 85–95# + 3 burpees
- TGU + Jumping Lunge + Run descending ladder (10/7/5/3 TGUs + 40/30/20/10 lunges with 400m runs between)

### Strength (named workouts)
- **Tongue Lashing**: Work to 1RM Press → 6 rounds 3× Press @ 75–80% → walking lunges → Curtis P's + rope climb → 30/30 jingle jangles
- **Riley Black**: 3–5× ladders (1–3 reps) with progressive loading on 3 bars (85%/90%/95%). Applies to: Bench, DL, Front Squat, OHS, Press, Squat, Weighted Pull-ups
- **Russian Aggression**: 1RM Front Squat → 10 rounds 3× FS @ 75% every 40 sec → 5 rounds DL @ 185# + sled push → 3 rounds pull-ups + push-ups
- **Olympic Day (Snatch Your Face / Gorilla Mania)**: Snatch/Power Clean/Jerk/Clean Pull/Front Squat at 80–90% in wave patterns (3r/2r/1r/2r)
- **Goldman Sachs**: 1RM Clean Squat & Push Press → EMOM work → KB swings + bench + tire drag
- **Honey Butter (Strength Maintenance)**: DL + box jumps + weighted sit-ups ascending load, heavy bag work, rope climb, DB crawl

### SMMF (Slow Massive Movement — high-volume)
Single massive effort challenges:
- 1 HSPU every 30 sec for 50 min (100 total)
- 100 get-ups @ 35# KB (15–20 min)
- 100-rep challenge with any big lift @ 50% 1RM
- 500m lunge (10×50m, varying carry methods every 50m)
- 100 burpee + pull-up combo
- 100× Curtis P's
- 100× sandbag get-ups
- 1-mile heavy bag carry

### Recovery
Easy effort 30–60 min at very low heart rate (−65% MHR): cycle, elliptical, hike, row, stair master, swim, walk

### Accumulation
Volume work with ladders and team formats:
- Sprint starts @ resistance band
- Team jerk ladder 5×(1–3) @ 75–80% BW
- Team TGU ladder @ 53# KB
- KB complexes (around the world, snatch, step-up)
- Fat bar DL + dead hang holds

### Density (EMOM)
- "Purple Church" — EMOM 10 min: 3 power clean + 3 hang squat clean + 3 push press
- "Slab City" — 20 rounds: 30-sec KB swing / 30-sec rest
- "Mad Dog 20/20" — 10 rounds: 30-sec jingle jangles / 30-sec rest
- "Thin Blue Line" — every 3 min for 6 rounds: 10 burpees + 25m bear crawl + 25m tire drag + 20 sit-ups

### Breathing Ladder
- Rep ladder 1–20 (or 1–20–1) with prescribed breaths between each rung
- Movements: ball slams, KB swing, KB snatch, KB two-hand clean & squat, thrusters
- Under 30 min = Power Endurance; over 30 min = Endurance under 90 min

### Power (Explosive)
- 1RM in: clean, push press, jerk, clean & jerk, snatch
- "Not Now Chief": 5 rounds 3× DL @ 90–95% + 3–6 depth jump & broad jump (5-min rest between)
- "Fuckin' Skanks": 5 rounds 3× weighted pull-up @ 90–95% + 3–6 clapping pull-ups (5-min rest)

### Power (Cardiovascular)
- 500m row under 1:30 + 400m run
- Litvinov conversion: 8× Front Squat @ 70% + 20m sled drag @ 100# (3 rounds)

### Endurance
Any cardiovascular activity 30–90 min: climb, cycle, hike, paddle, row, run, ski, swim

### Job Specific / Sport Specific
Named CrossFit workouts + original Horsemen workouts used as sport-specific conditioning

### Circuits
- **Short**: Elizabeth (21-15-9 Clean + Ring Dips), Grace (30× C&J @ 135#), Fran (21-15-9 Thrusters + Pull-ups), Helen, Jackie, Annie
- **Medium/Long**: Chelsea (EMOM 30 min: 5 pull-ups + 10 push-ups + 15 squats), PBR (AMRAP 20 min: 10 FS + 10 pull-ups + 10 weighted sit-ups + 10 push-ups)
- **Complementary**: Horses Ass (30-20-10 DL + HSPU), Jonestown Sprint, Jonesworthy

### Focus Circuits (Core/Upper Body)
- "Appeal to God": 3 rounds of 30-sec holds (GHD sit-up, V-sit, ring static hold, tuck hang, plank variations) with 5 perfect push-ups during each rest
- "Slam Your Balls" (30-20-10): ball slam + KB step-up @ 2×35# on 20" box

## Key Exercises & Definitions

| Name | Description |
|------|-------------|
| Curtis P | Hang Squat Clean → Lunge R → Lunge L → Push Press (all at same weight) |
| Leg Blaster | 20× squat + 10× lunge each leg + 10× jumping lunge each leg + 10× squat jump |
| Mr. Spectacular | 2-hand KB clean/press → walk hands out to push-up → push-up → walk back up |
| Renegade Man Maker | Power clean + press → push-up position → push-up → row each arm |
| Barbell Complex | DL + Row + Hang Clean + Front Squat + Push Press + Back Squat + Push-up (6 each) |
| Double Eagle | Clean sandbag to shoulder + 10 deep squats + tire drag forward 40ft + drag back |
| Slasher to Halo | KB drop to hip → slash across body → around back of head → other shoulder |
| Jingle Jangles | 5-yard back-and-forth sprint touching ground at each cone |
| The Exercise | Snatch + OHS (without lowering weight between) — Dan John |
| SDHP | Sumo Deadlift High Pull |
| Half Moon | Sandbag: reach over body, pick up, arc high overhead, place down other side |
